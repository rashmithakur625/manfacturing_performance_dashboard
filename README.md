Project Overview:
The Manufacturing Performance Dashboard is a comprehensive data visualization solution developed using Microsoft Power BI. This project integrates multi-dimensional manufacturing data—including production metrics, quality control, maintenance schedules, and operational costs—into a single, interactive interface.

Project Objective
The goal is to provide plant managers and stakeholders with a high-level view of floor operations. By analyzing machine efficiency and production output, the dashboard identifies bottlenecks and supports strategic decision-making to improve overall productivity.

 Technical Implementation:
Tools Used
Microsoft Power BI: Used for end-to-end data modeling, DAX calculations, and interactive report design.

Microsoft Excel: Used for data structuring, cleaning, and initial organization of the manufacturing datasets.

Data Modeling:
Schema: Implemented a Star Schema for optimal performance.

Date Dimension: A centralized DimDate table was created to enable advanced time-intelligence analysis (Monthly, Quarterly, and Yearly trends).

Data Cleaning: Leveraged Power Query to ensure data consistency and accuracy across Production, Quality, and Cost tables.

 Dashboard Features:
 
The report is structured into three specialized analytical views:

Production Analysis: Tracks output volume vs. targets and identifies efficiency gaps.

Quality & Maintenance: Monitors machine downtime, maintenance cycles, and product defect rates.

Cost & Efficiency: Analyzes operational expenditures and resource utilization to identify cost-saving opportunities.
